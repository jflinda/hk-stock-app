# 安装 HK Stock App APK 到手机

## 📱 快速安装指南

### APK 文件位置：
```
c:\Users\jflin\WorkBuddy\20260329125422\stocktrading\frontend\build\app\outputs\apk\debug\app-debug.apk
```

### 安装步骤：
1. **连接手机**：通过 USB 线连接 Android 手机到电脑
2. **选择传输模式**：在手机上选择"文件传输"或"传输文件"（MTP模式）
3. **复制文件**：
   - 双击打开手机的存储（在"此电脑"或"我的电脑"中）
   - 导航到 `Downloads` 文件夹
   - 将 `app-debug.apk` 文件从电脑复制到此文件夹
4. **在手机上安装**：
   - 断开 USB 连接或切换为仅充电模式
   - 打开手机的文件管理器
   - 找到 `Downloads/app-debug.apk`
   - 点击安装
5. **开启未知来源**（如需要）：
   - 如果提示"禁止安装未知来源应用"
   - 进入手机设置 → 安全/隐私 → 未知来源
   - 允许从"文件管理器"安装应用

### 🔧 启动前准备：
1. **确保后端运行**：在电脑上，API 服务已在运行（http://localhost:8000）
2. **同一网络**：手机和电脑需要在同一 Wi-Fi 网络
3. **启动应用**：安装完成后，在手机上找到 "HK Stock App" 图标并点击打开

### 📊 验证安装成功：
- ✅ 应用图标出现在主屏幕
- ✅ 应用可以正常打开
- ✅ 能看到加载界面和市场数据

### ⚠️ 常见问题：
1. **安装失败**：检查存储空间是否足够（>200MB）
2. **解析包错误**：重新下载 APK 文件
3. **无法连接后端**：确保电脑防火墙允许 8000 端口
